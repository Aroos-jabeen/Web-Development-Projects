# Pakistan City Explorer
A responsive, interactive web-based gallery showcasing major urban centers in Pakistan. This project demonstrates modern CSS layout techniques and DOM manipulation using Vanilla JavaScript.
## 🚀 Technical Overview
This project focuses on Glassmorphism UI and State Management (toggling views between a landing placeholder and dynamic content).
Key Features
• Dynamic UI Updates: JavaScript-driven image and text swapping without page reloads.
• Modern CSS: Implementation of backdrop-filter for frosted glass effects and flexbox for layout alignment.
• Interactive States: Visual feedback on hover and active selection (using "Viewing" status indicators).
• Responsive Design: Utilization of object-fit: cover to maintain image aspect ratios across different screen sizes.
## 🛠️ Tech Stack
• Frontend: HTML5, CSS3
• Scripting: JavaScript (ES6+)
• Design Pattern: Glassmorphism & Minimalist Dark Mode
